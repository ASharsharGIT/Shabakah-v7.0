## 🎨 Customization Instructions

### 1. **Update Brand Colors**

Edit `assets/css/main.css` (lines 30-45):

```css
:root {
    /* Extract colors from your actual logo */
    --primary-navy: #0A1628;   /* Your dark blue */
    --accent-blue: #3B82F6;    /* Your brand blue */
    --accent-cyan: #06B6D4;    /* Your secondary */
    --accent-gold: #F59E0B;    /* Saudi gold */
}
```

**How to extract logo colors:**
1. Open logo in Figma/Photoshop
2. Use eyedropper tool
3. Copy HEX codes
4. Replace CSS variables

### 2. **Add Your Logo**

Create 3 files in `assets/images/`:
- `shabakah-logo.svg` (main logo, colored)
- `shabakah-logo-white.svg` (white version)
- `favicon.svg` (browser icon)

Then update these lines in HTML:
```html
<!-- Header logo -->
<img src="assets/images/shabakah-logo.svg" alt="SHABAKAH">

<!-- Footer logo -->
<img src="assets/images/shabakah-logo-white.svg" alt="SHABAKAH">
```

### 3. **Update Company Info**

Search and replace in ALL HTML files:

```javascript
// Phone
Find: "+966 XX XXX XXXX"
Replace: "+966 11 XXX XXXX" // Your real number

// Email  
Find: "info@shabakah.sa"
Replace: "contact@shabakah.sa" // Verify actual email

// Address
Find: "Riyadh, Saudi Arabia"
Replace: "Building Name, Street, District, Riyadh 12345" // Full address
```

### 4. **Add Icons (Important!)**

**Current status**: Using emoji placeholders

**Solution**: Add Font Awesome CDN

Add to `<head>` of ALL HTML files:
```html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
```

Then replace icon classes:
```html
<!-- OLD -->
<i class="icon-search"></i>

<!-- NEW -->
<i class="fas fa-search"></i>
```

**Icon mapping**:
- `.icon-search` → `.fas .fa-search`
- `.icon-shield` → `.fas .fa-shield-alt`
- `.icon-datacenter` → `.fas .fa-server`
- `.icon-security` → `.fas .fa-lock`
- `.icon-linkedin` → `.fab .fa-linkedin`
- `.icon-twitter` → `.fab .fa-twitter`

Full list in `assets/css/icons.css`
