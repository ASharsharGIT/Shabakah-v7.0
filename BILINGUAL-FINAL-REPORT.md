### 3. Language Toggle Functionality

#### How It Works:
```javascript
1. User clicks language toggle button (EN | العربية)
2. System detects current language
3. Switches to opposite language (en ↔ ar)
4. Saves preference to localStorage
5. Reloads page with new language
6. Updates:
   ✅ HTML lang attribute (lang="en" or lang="ar")
   ✅ HTML dir attribute (dir="ltr" or dir="rtl")
   ✅ Body classes (class="ltr" or class="rtl")
   ✅ All text content (complete translations)
   ✅ Font family (Inter or Tajawal)
   ✅ Layout direction (LTR or RTL)
```

#### Persistence:
- Language preference saved to browser localStorage
- User returns to chosen language on next visit
- Works across all pages seamlessly
